# fkandu_manager_bot

Telegram-бот-менеджер для канала **«Дети и Желания»** ([@fkandu](https://t.me/fkandu)). Собирает заявки на рекламу от потенциальных партнёров, сохраняет данные в Google Sheets и отправляет уведомление администратору.

## Функционал

- Приветственное сообщение с описанием канала
- Пошаговый опрос (FSM) через inline-клавиатуры:
  1. Категория продукта
  2. Описание товара (свободный текст или ссылка)
  3. Бюджет на размещение
  4. Сроки запуска
- Автоматическая оценка лида: горячий / тёплый / холодный
- Запись заявки в Google Sheets
- Уведомление администратора в личные сообщения

## Технологии

- Python 3.9+
- [aiogram 3](https://docs.aiogram.dev/) — асинхронный Telegram Bot API
- [gspread](https://docs.gspread.org/) — работа с Google Sheets
- [python-dotenv](https://github.com/theskumar/python-dotenv) — управление переменными окружения

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```
BOT_TOKEN=your_telegram_bot_token
ADMIN_ID=your_telegram_user_id
SPREADSHEET_ID=your_google_spreadsheet_id
```

### 3. Google Sheets

Поместите файл `credentials.json` (сервисный аккаунт Google) в корень проекта. Сервисный аккаунт должен иметь доступ на редактирование к таблице, указанной в `SPREADSHEET_ID`.

При первом запуске бот автоматически создаст заголовки в таблице.

### 4. Запуск

```bash
python3 bot.py
```

## Оценка лида

| Оценка | Условия |
|--------|---------|
| Горячий 🔥 | Бюджет ≥ 50 000 ₽ **или** бюджет ≥ 30 000 ₽ + запуск на этой неделе |
| Тёплый 💛 | Бюджет ≥ 10 000 ₽ + запуск в ближайшие 2 недели / месяц |
| Холодный 🧊 | Все остальные случаи |
